# QiYuan OS - 启元操作系统

!https://github.com/SYSTEM-WIN-ZDY/QiYuanOS/edit/main/QiYuan%20OS.png

## 项目概述
QiYuan OS是一个教育性质的操作系统项目，旨在实现基本的操作系统功能，包括：
- 内核基础功能
- 文件系统支持
- 图形用户界面

## 目录结构
- `kernel/` - 内核代码
- `fs/` - 文件系统实现
- `gui/` - 图形界面代码
- `drivers/` - 驱动程序
- `boot/` - 引导代码
- `tools/` - 开发工具

## 构建说明
使用GCC工具链进行编译构建。
